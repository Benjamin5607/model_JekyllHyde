Jekyll & Hyde — Install Package v1.0.0
======================================

All release files use maximum compression (gzip / ZIP deflate level 9).

1. Download from GitHub Releases (v1.0.0):
   - JekyllHyde-1.0.0-app.zip
   - JekyllHyde-1.0.0-model.part00.gz
   - JekyllHyde-1.0.0-model.part01.gz
   - JekyllHyde-1.0.0-model.part02.gz

2. Extract app.zip to e.g. C:\JekyllHyde
3. Copy the three .gz model parts into the same folder
4. Run install.bat (decompresses + merges model, sets up venv)
5. Open http://127.0.0.1:8080

Optional local build (single compressed archive):
  scripts\build_release.ps1
  -> dist\JekyllHyde-1.0.0-win64.zip (shows live progress during compression)

Support: https://github.com/Benjamin5607/model_JekyllHyde
