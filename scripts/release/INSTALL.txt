Jekyll & Hyde — Install Package v1.2.0
======================================

1. Download from GitHub Releases (v1.2.0):
   - JekyllHyde-1.2.0-app.zip
   - JekyllHyde-1.2.0-model.part00.gz
   - JekyllHyde-1.2.0-model.part01.gz
   - JekyllHyde-1.2.0-model.part02.gz

2. Extract app.zip to e.g. C:\JekyllHyde
3. Copy the three .gz model parts into the same folder
4. Run install.bat
5. Open http://127.0.0.1:8080
